# AegisOS

A simple educational operating system built from scratch.

## Features
- Bootloader (x86 Assembly)
- Kernel (C)
- VGA Text Output
- Keyboard Input
- Shell (commands)
- Memory Management (Heap)
- Interrupt Handling (IDT)
- Basic Multitasking
- Paging (basic)
- Simple File System (ls)

## Commands
help - list commands  
clear - clear screen  
mem - memory test  
task - run multitasking demo  
ls - list files  
about - project info  
version - OS version  

## How to Run
make clean
make
make run
